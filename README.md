dotfiles
=========

```sh
$ mkdir ~/dev
$ cd ~/dev/
$ git clone https://github.com/sile/dotfiles.git
$ cd dotfiles/

$ ./install.sh
$ emacs --script install.el
$ ./setup.sh
```
